## Why Perplexity is uniquely useful for citation.is

### 1. **Perplexity is built around citations**
- Perplexity was the **first generative AI tool to add citations** to its responses. [tcsedsystem.libguides](https://tcsedsystem.libguides.com/ai_tools_for_lit_discovery/perplexity)
- Every Perplexity answer includes **numbered citations** with clickable links to original sources. [youtube](https://www.youtube.com/watch?v=h7Y6e9Ae1ns)
- This makes Perplexity the **ideal discoverability engine** for sites like citation.is that provide verified claims.

### 2. **Perplexity prioritizes authoritative, extractable sources**
Perplexity’s citation system favors pages that have:
- **PerplexityBot crawler access** (no login walls, no heavy scripts)
- **Very high recency weight** (fresh content, e.g., updated regularly)
- **Authority signals** (UniProt, NCBI, PubMed cross-references—as you already have)
- **Extractable structured data** (FAQPage, JSON-LD, clear headings) [solcrys](https://solcrys.com/optimize-for-perplexity/)

Your site already has **recency** (updated 2026-06-10), **authority** (UniProt/NCBI/PubMed), and **open data** (CC BY 4.0, public API). Adding schema would make it even more citation-ready. [citation](https://citation.is)

### 3. **Perplexity API is developing citation support**
- Perplexity’s API now supports **grounded LLM responses** with citations. [blog.hypertxt](https://blog.hypertxt.ai/2024/11/08/perplexity-api-citations/)
- Citations are under active development for JSON responses, but the platform is designing for **direct source linking, citation metadata, and real-time verification**. [blog.hypertxt](https://blog.hypertxt.ai/2024/11/08/perplexity-api-citations/)
- This means future API integrations could let external apps (like your MCP server) **feed verified claims into Perplexity answers**.

### 4. **Perplexity can be used as a discovery channel**
You can use Perplexity to:
- **Find researchers, journalists, and publishers** who cite scientific claims in your domains.
- Ask: “Who is researching salmon aquaculture parasite loads in 2026?” → Perplexity returns papers, authors, labs.
- Then outreach to them to use citation.is for verification.

## How to integrate citation.is with Perplexity
